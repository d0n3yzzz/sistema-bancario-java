Sistema Bancário em Java

Este projeto é um sistema bancário simples desenvolvido em Java com o objetivo de praticar conceitos de Programação Orientada a Objetos (POO) e lógica de programação.

A aplicação funciona via terminal (CLI) e permite realizar operações básicas de um banco.



Funcionalidades

 Criar conta bancária
 Depositar valores
 Sacar valores
 Consultar saldo
 Persistência de dados em arquivo (`contas.txt`)



Tecnologias utilizadas

- Java
- Programação Orientada a Objetos (POO)
- ArrayList
- FileWriter e BufferedReader
